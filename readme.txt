<概要>
大学で提供された実機でCUIシューティングゲームを実装。
公開しているフォルダ内だけではコンパイルは出来ない。
----------------------------------------------------------------------

<game.cについて>
・＊オブジェクトのy座標の初期化位置をずらすことでオブジェクトがすべて同時に落ちないように実現している。

・描写スピードの調整では1ループにオブジェクトと自機一マスずつ動かすようにし、
自分で作ったtime_c関数によって動く速さを自由に変えられるようにしている。

・自機の移動は方向x座標を記録する変数をもつ構造体を作成し、入力があった場合、
方向のみを更新し、慣性で動かす場合方向の入力をよんでx座標を動かすという操作で慣性を実現している。
スタートから入力がない場合、その場に留まるように実装している。

・自機、＊オブジェクトの移動に関して、locate関数で移動は実現できるがその軌跡まで表示してしまう。
これを解消するため、描写(移動)、表示時間調整、一つ前の移動跡の消去という手順で、
移動に関しての実装を行った。

・当たり判定は、終了のフラグを用意して、このフラグが立たなければ
ゲーム続行するようにwhileでループを回し続けている。

・スコアは＊オブジェクトが最下点に着いた時、一つに着き+5点追加されるように実装した。
